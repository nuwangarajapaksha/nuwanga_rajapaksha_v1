NDCamera – E-Commerce & M-Commerce Camera Selling Platform

My Role: Full-Stack Software Engineer (Solo Project)
Technology Stack: Java, Java EE, MySQL, Hibernate, JPA, JavaScript, HTML, CSS, Bootstrap, JSON, Axios, Android, Firebase, Google Authentication, Mail Sender Service, PayHere Payment Gateway, JUnit

Project Overview

NDCamera is a comprehensive camera-selling platform developed as a final year university project. The solution consists of two integrated components: a web-based e-commerce application and a mobile m-commerce Android application, both designed to deliver a user-friendly and feature-rich online shopping experience for camera products.

The platform includes a customer-facing interface with modern shopping functionalities as well as an admin panel for complete store management. The solution provides real-time product search, order tracking, secure checkout, and personalized user experiences. Both the web and mobile apps are developed independently and function as complementary systems.

Architecture

The system is designed with two separate components:

  • E-Commerce Web Application: Built using Java EE, this component includes a JSP/Servlet-based MVC architecture for backend services, integrates MySQL for database operations, and uses Hibernate and JPA for ORM. The frontend is built using HTML, CSS, JavaScript, and Bootstrap for responsiveness, while JSON and Axios enable efficient client–server communication. The application also includes a built-in mail sender service for order confirmations and admin notifications, along with PayHere Payment Gateway integration for secure online transactions.
  • M-Commerce Mobile Application: Developed using Android (Java) and integrated with Firebase Realtime Database. Google Authentication Services are used to enable quick and secure login functionality, and PayHere Payment Gateway is integrated to support secure in-app payments.

Key Features

User Side (Both Web & Mobile):

  • User registration and login (manual & Google sign-in for mobile) with Profile
  • Product search and category-wise filtering
  • Product view with details and images
  • Add to cart and wishlist
  • Secure checkout with PayHere Payment Gateway and email confirmation
  • Order history tracking
  • Blog access and contact details

Admin Side (Web Only):

  • Admin profile and authentication
  • Product and brand management
  • Order management with automated email notifications
  • User management
  • Business and store settings configuration
