PAMUNUWILA HARDWARE (Pvt) Ltd – Desktop-Based Hardware Shop Management System

My Role: Full-Stack Software Engineer
Technology Stack: Java, Java Swing, MySQL, Jasper Reports, JUnit

Project Overview

PAMUNUWILA HARDWARE is a comprehensive desktop-based hardware shop management system developed for PAMUNUWILA HARDWARE (Pvt) Ltd. This was an individual project I undertook for my 1st-year final project at university. The system was designed to digitize and streamline all operational activities of the hardware business, from user and inventory management to sales, delivery, and financial reporting.

The solution serves as an all-in-one tool for the daily operations of the business, providing seamless management of products, stock levels, purchases, sales, deliveries, and customer interactions, while maintaining a centralized, secure database. It replaces manual recordkeeping with a highly organized, automated, and scalable system.

Architecture

  • This desktop-based application is built using Java and Java Swing for the frontend user interface, backed by MySQL for structured data storage. Jasper Reports is used for generating invoices, bills, and all historical reports, including daily, monthly, and yearly summaries.
  • The system is modularly designed, allowing each functional domain to operate independently yet cohesively as part of the overall system.

Key Features

  • Login, registration, and secure user management
  • Create and manage user roles and access levels
  • Store and manage supplier details and associated companies
  • Add and track items, real-time stock updates, low stock alerts
  • Manage incoming goods and update stock accordingly
  • Handle transactions, customer records, and purchase history
  • Track delivery orders and manage transportation logistics
  • Secure data backup and restoration for disaster recovery
  • Generate and print all necessary documents using Jasper Reports
  • Access past transactions and analytics for business insights

My Role & Contributions

As the sole developer of this project, I was responsible for the entire development lifecycle:

  • Conducted requirement analysis, created database schema, and defined the modular architecture
  • Built the complete desktop UI using Java Swing, focusing on user experience and ease of use
  • Developed all business logic, database interaction modules, and integrated MySQL for persistent data storage
  • Created dynamic, printable invoices and reports using Jasper Reports for sales, inventory, deliveries, and backups
  • Performed unit and integration testing across modules to ensure a bug-free and reliable application
  • Used JUnit for unit testing and ensured robustness through extensive debugging and validation
  • Delivered the project with full technical documentation and deployment-ready setup
  • utilized my leadership, critical thinking, problem-solving, and situation-handling skills to manage the entire development process efficiently, overcome challenges, and deliver a fully functional, professional-grade solution.

Clients & Implementation

This project was successfully deployed at PAMUNUWILA HARDWARE (Pvt) Ltd and has been actively used to manage their daily business operations. In addition to serving a real client, it was also submitted and presented as my final project for the first year of my university studies. This dual-purpose project provided valuable real-world experience in understanding client needs, delivering production-ready software, and ensuring customer satisfaction. It significantly strengthened my practical development skills and my ability to independently deliver a full-scale commercial application—bridging the gap between academic knowledge and professional-level software engineering.
