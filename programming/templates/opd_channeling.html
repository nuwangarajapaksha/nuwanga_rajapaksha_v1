Project Overview

OPD Channeling is a fully functional E-Channeling Web Platform developed as a solo second-year university assignment. The system is designed to digitize and streamline outpatient department (OPD) appointment management by connecting patients, doctors, and hospitals in a single online platform.

The main focus of the application is to manage e-channeling operations, including patient handling, doctor handling, and hospital handling. Patients can register and book appointments with available doctors. Doctors can register to offer their consultation services, manage their channels, and view upcoming appointments. Hospitals can pre-register in the system, allowing their doctors and services to be listed. An integrated admin panel allows administrators to manage patients, doctors, hospitals, and all appointment-related operations, ensuring smooth coordination between stakeholders.

Architecture

  • The application is built using a PHP backend for server-side logic, combined with HTML, CSS, JavaScript, and Bootstrap for a responsive and intuitive frontend. MySQL serves as the relational database, storing user profiles, hospital details, appointment records, and administrative data. The application follows a modular structure to separate concerns between the core functionalities of patients, doctors, hospitals, and the admin panel.

Key Features

Patient Side:

  • Registration, secure login, and profile
  • E-channeling functionality to book doctor appointments
  • View appointment history and details

    Doctor Side:

  • Registration, secure login, and profile
  • Dashboard to view and manage booked appointments (channels)
  • Manage availability for patient bookings

    Admin Side:

  • Secure registration, login, and profile
  • Hospital registration and management
  • Manage patient, doctor, and hospital records
  • Manage appointments and schedules
