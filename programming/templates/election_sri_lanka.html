Project Overview

Election Sri Lanka is a fully functional Online Election Management Platform developed as a solo second-year university assignment. The system is designed to digitize and streamline the entire election process, from voter registration to secure voting and automatic result generation.

The platform enables voter management, candidate management, party management, and position handling, ensuring a smooth and transparent election workflow.Voters can register and login using their National Identity Card (NIC) number and cast votes for their preferred parties and candidates. The system also features an integrated guideline page to assist users.

On the administrative side, admins can register and log in to manage all aspects of the election, including administrators, candidates, parties, positions, voters, and results. The result computation process is automated, eliminating the possibility of human error and increasing election credibility.

Architecture

  • The application is built using a PHP backend for server-side processing, integrated with MySQL as the relational database to store voter records, election data, and results. The frontend leverages HTML, CSS, JavaScript, and Bootstrap to provide a responsive, user-friendly interface. The system follows a modular approach to separate core functionalities for voters and administrators, ensuring maintainability and scalability.

Key Features

Voter Side:

  • NIC-validated registration and login with user profile
  • Cast votes for parties and candidates
  • Access to a guideline page for election instructions

    Admin Side:

  • Registration, secure login, and profile
  • Manage administrators, candidates, parties, positions, and voters
  • Add candidates and parties to the system
  • Automatic result generation after voting ends
